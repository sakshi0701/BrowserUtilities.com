# BrowserUtilities.com



[Visit Website](https://let-us-code.herokuapp.com/login "Welcome to BrowserUtilities.com")

## To run the application:

`npm i` <br />
`npm start`

## Features:

**A user can:** <br />
- Scan QR Code  <br />
- Generate QR Code </br>
- Scan QR Code through camera </br>
- Generate passwords </br>
- Shorten URLs.
